Place the official CHERNO GAMING theme song MP3 in this folder as an owner/deploy-time asset.

Visitors should only get one mute/unmute toggle for the configured track. Do not
add play/stop controls, a public upload, a picker, or a URL field for changing
the MP3.

Example:

```js
themeSong: {
  title: "CHERNO GAMING Theme Song",
  src: "/assets/audio/theme-song.mp3",
  loop: true,
}
```
